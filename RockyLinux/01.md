# ファイル

## ディレクトリを移動するコマンド
```shell
cd [ディレクトリ名]
```

### カレントディレクトリ　→　？
#### ホームディレクトリ
```shell
cd
```
```shell
cd ~
```
#### 親ディレクトリ
```shell
cd ..
```
#### ルートディレクトリ
```shell
cd /
```

## 現在のディレクトリ
```shell
pwd
```
結果：
```shell
[game@localhost ~]$ pwd
/home/game
```

## ディレクトリの内容を表示するコマンド
```shell
ls
```
```shell
[game@localhost ~]$ ls
Desktop 
```

### オプション
* ls -a ：隠しファイルを表示する
```shell
[game@localhost ~]$ ls -a
. .. .bash_logout　Desktop 
```
* ls -l ：長形式で表示する
```shell
```

### カレントディレクトリ以下のファイル・ディレクトリをサブディレクトリ内も含めすべて長形式で表示
ls -laR
### /varディレクトリ以下のファイル・ディレクトリをサブディレクトリ内も含めすべて表示
