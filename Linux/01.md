# ファイル

## ディレクトリを移動するコマンド
```shell
cd [ディレクトリ名]
```
### カレントディレクトリ　→ ホームディレクトリ
```shell
cd
```
### カレントディレクトリ → 親ディレクトリ
```shell
cd ..
```
### カレントディレクトリ → ルートディレクトリ
```shell
cd /
```

## ディレクトリの内容を表示するコマンド
```shell
ls
```
