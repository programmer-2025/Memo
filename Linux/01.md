# ファイル

## ディレクトリを移動するコマンド
```shell
cd [ディレクトリ名]
```
### カレントディレクトリ　→ ホームディレクトリ
```shell
cd
```
### カレントディレクトリ → 親ディレクトリ
```shell
cd ..
```
### カレントディレクトリ → ルートディレクトリ
```shell
cd /
```

## 現在のディレクトリ
```shell
pwd
```
結果：
```shell
[game@localhost ~]$ pwd
/home/game
```

## ディレクトリの内容を表示するコマンド
```shell
ls
```
